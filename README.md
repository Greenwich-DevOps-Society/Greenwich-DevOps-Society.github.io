## Greenwich DevOps Society Website

Official website repository for Greenwich DevOps Society.

#### About The Society

We run varied group projects, practical challenges, and social activities to help members build technical skills, confidence, and employability.

Our goals are to:

- Build real-world DevOps and software engineering experience.
- Learn through collaboration, workshops, and project work.
- Encourage networking and community across students.
- Create an inclusive environment where everyone can grow.

We also host a weekly women-in-tech meeting (led by a woman on the course)
to support connection, learning, and mentorship.

#### What We Do

- Group coding and DevOps challenges.
- Student-led project work.
- Visits to computer science and tech events.
- Skill-building sessions and peer support.
- Social meetups to help members make friends and feel part of the community.
